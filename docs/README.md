# 기능 목록 정리

## 첫화면

-  [ ] 소셜 로그인 진행
-  [ ] 로그인 성공시 메인화면 이동
-  [ ] 로그인 실패시 실패 알림 생성

## 메인화면

-  [ ] 사용자 왼쪽 창에 보여주기
-  [ ] 글 생성하기 버튼
   -  [ ] 생성 창 열리기
-  [ ] 글 목록 보이기
   -  [ ] 글 제목, 내용, 수정, 삭제 버튼
   -  [ ] 스크롤로 쌓아지도록

## 글 작성화면

-  [ ] 글 제목 입력
-  [ ] 글 내용 입력
-  [ ] 취소 버튼
   -  [ ] 메인화면으로 이동
-  [ ] 생성 버튼
   -  [ ] 글 목록에 쌓아지고 메인화면으로 이동

## 글 수정화면

-  [ ] 글 제목 입력
-  [ ] 글 내용 입력
-  [ ] 취소 버튼
   -  [ ] 메인화면으로 이동
-  [ ] 수정 버튼
   -  [ ] 글 수정되면서 메인화면으로 이동
